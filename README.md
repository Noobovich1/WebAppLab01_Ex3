# WebAppLab01_Ex3
Booking website of Hue Citadel
https://github.com/Noobovich1/WebAppLab01_Ex3

## Site Map
This project consists of [7] HTML pages:
1. **index.html**: Homepage
2. **history.html**
3. **attractions.html**
4. **tours.html**
5. **gallery.html**
6. **map.html**
7. **booking.html**

## How to Run Locally
1.  Download or Clone this repository.
2.  Open `index.html` in any web browser (Recommend using live server extension).
3.  *Note:* To test the Dark Mode persistence, ensure cookies/local storage are enabled.
